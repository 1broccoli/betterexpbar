Version 2.9.5 - January 22, 2026
- Added per-character profile management system using AceDB
- Added AceDBOptions-3.0 for profile management interface
- Each character can now create, copy, delete, and switch between multiple profiles
- Profile options accessible through minimap button and Blizzard options panel
- Changed from SavedVariables to SavedVariablesPerCharacter in TOC file
- Each character maintains independent profile sets for bar positions, colors, and preferences
